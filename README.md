# MapTest_2

Для запуска приложения:
1. Клонировать репозиторий локально
2. cd MapTest_2
3. cd frontend
4. npm install
5. cd ../server
6. npm install
7. npm start

запустится сервер с данными на порту 8082

В другои окне открыть директорию frontend

8. npm run dev

приложение автоматически запустится на порту 8080 или 8081
